## File review website

1. User can create a review with file title and rating stars using the HTML form. After the review is created, reload the page to render all of the review in the HTML table.

2. User can change the rating score to update the rating of each review. After the rating score is changed, reload the page to render updated data again.

3. Be sure to set the security rule in the firestore dashboard, making visitors have the right to read, create and update data in `reviewList` collection.

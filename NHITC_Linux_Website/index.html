Linux 是什麼?

Linux是一種自由和開放原始碼的類UNIX作業系統。該作業系統的核心由林納斯·托瓦茲在1991年10月5日首次發布，在加上使用者空間的應用程式之後，成為Linux作業系統。

Debian 發行版

Debian is the best Linux OS.

Debian以其堅守Unix和自由軟體的精神，以及其給予使用者的眾多選擇而聞名。現時Debian提供了超過25,000個軟體，超過50,000個軟體套件，並正式支援10個電腦系統結構。
